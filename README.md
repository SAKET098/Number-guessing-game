# 🎯 Number Guessing Game

This is just a small number guessing game to practise basic python 


A simple command-line Number Guessing Game built with Python.

The computer randomly generates a number between **1 and 100**, and the player has **8 attempts** to guess it correctly. After each incorrect guess, the game provides a hint indicating whether the guess was **too high** or **too low**.

---

## 🚀 Features

* 🎲 Random number generation using Python's `random` module
* 🔢 Guess a number between **1 and 100**
* ❤️ 8 attempts to win
* 📈 Hint when the guess is too high
* 📉 Hint when the guess is too low
* 🎉 Win message when the correct number is guessed
* 😢 Lose message when all attempts are exhausted
* ▶️ Option to start or quit the game before playing
